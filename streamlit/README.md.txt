# Snowflake Ecommerce Project

This project implements an end-to-end Snowflake pipeline for ecommerce data.

## Components
- Ingestion using Snowflake stage and COPY INTO
- Transformation using UDF, streams, stored procedures, tasks, and dynamic tables
- Delivery using Streamlit in Snowflake

## DevOps
- Source control using GitHub
- SQL scripts version-controlled for change tracking
